===========================================
systemd_networkd role for OpenStack-Ansible
===========================================

This role will configure systemd-networkd link, network, and netdev files.

Documentation for the project can be found at:
  https://docs.openstack.org/ansible-role-systemd_networkd/latest

  https://opendev.org/openstack/ansible-role-systemd_networkd

The project home is at:
  https://launchpad.net/openstack-ansible

The project bug tracker is located at:
